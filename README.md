# 新店 ADT 预测器（初版）

#项目简介
本项目基于某咖啡连锁品牌门店的历史经营数据，利用机器学习方法构建了一个新店日均交易量（ADT）预测模型。输入新店的基本特征（渠道、城市等级、面积、租金等），即可预测开业稳定后的日均交易次数，为选址决策提供数据支持。
目前模型在测试集上的 R² 达到 0.62，能够解释超过 60% 的 ADT 变异，在同类静态特征预测中效果良好。本应用以 Streamlit 搭建交互界面，方便业务人员直接使用。
注：本版本为初版，后续将持续优化模型、增加更多特征、完善业务逻辑。

#主要功能
选择渠道（如交通枢纽、社区型商业等）
选择城市等级（T1 ~ T5）
输入门店面积（平方米）
输入租金（支持两种方式：月租金总额 或 每平米月租金）
一键预测日均交易数（ADT），并给出结果

#安装依赖

#模型说明
训练数据：某连锁咖啡品牌 7696 家稳定门店（开业≥6个月）的截面数据（FY26 P01 期）
特征：渠道、城市等级、面积、月租金

#目标：日均交易次数（ADT）
算法：XGBoost / LightGBM，经交叉验证 R² 约为 0.62

#预处理：类别特征独热编码，数值特征标准化
