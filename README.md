# 某连锁咖啡品牌渠道画像分析报告（FY26 P01）

## 项目简介
本项目基于某连锁咖啡品牌 FY26 P01 的截面数据，分析门店渠道与城市等级对门店日均交易量（ADT）的影响，为开店策略提供数据支持。分析包含数据清洗、描述性统计、方差分析（ANOVA）、事后检验（Tukey HSD）、双因素交互作用分析、相关性分析等。

## 数据说明
- 数据源：database FY26 P01 截面数据，包含 7991 家在营门店（开业满3个月后稳定门店 7696 家）
- 关键字段：ADT（日均交易数）、AT（客单价）、面积、TZ（商圈/渠道）、城市等级（T1~T5）

## 分析内容
1. **数据清洗与预处理**  
   - 筛选稳定门店（开业≥3个月）  
   - 处理异常值，创建坪效、渠道映射、面积分组等特征

2. **描述性统计分析**  
   - 各渠道 ADT 对比（均值、中位数、标准差）  
   - 渠道箱线图与柱状图  
   - 城市等级×渠道交叉分析（分组箱线图与柱状图）

3. **统计验证**  
   - 整体 ANOVA：渠道间 ADT 差异显著（F=151.77, p≈0）  
   - Tukey HSD 事后检验：交通枢纽店显著高于所有其他渠道  
   - 城市等级分层 ANOVA：T1~T5 各等级内渠道差异均显著，但 F 值随等级下降而减小  
   - 双因素 ANOVA（渠道×城市等级）：交互作用不显著（p=0.226），但图形显示交通枢纽店对城市等级敏感度高

4. **面积对 ADT 的影响**  
   - 面积分组 ANOVA：组间差异显著（F=125.3, p≈0）  
   - Tukey 检验：大面积店 ADT 显著高于中、小面积店  
   - Pearson 相关：面积与 ADT 正相关，但 R² 较低，非唯一决定因素

## 结果摘要
- **渠道是关键因素**：交通枢纽店 ADT 遥遥领先，但内部差异大；市级商业中心、区域级商业区是稳健的高客流渠道；社区店、住宅、办公商圈业绩相近，属第二梯队。
- **城市等级影响渠道效应**：高线城市渠道差异大，低线城市差异小。交通枢纽店在高线城市优势明显，在低线城市优势减弱。
- **面积与 ADT 正相关**，但解释力有限，需结合渠道和城市等级综合考虑。

## 业务建议
- **新店选址**  
  - T1、T2 城市优先布局交通枢纽店（高回报，但需评估风险）  
  - T3~T5 城市优先考虑市级商业中心、区域级商业区（稳定高客流）  
  - 社区店、住宅店可作为填充型门店，控制面积以优化坪效
- **存量店优化**  
  - 对交通枢纽中的低业绩门店深入分析原因（面积、位置、竞品等）  
  - 对社区店中面积过大的门店评估坪效，考虑优化
